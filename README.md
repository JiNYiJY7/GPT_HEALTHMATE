# 📌 GPT HealthMate  

A health assistant AI built with **Streamlit + Python**, supporting BMI calculation, health tips, and GPT-powered conversations.  

---

## 🚀 Features  
- 🔐 User Registration & Login  
- 💬 Chat with GPT Health Assistant  
- 📊 BMI Calculation with Health Insights  
- 🗄️ SQLite Database Storage  

---

## 🛠️ Installation  

1. **Clone the repository**  
   ```bash
   git clone https://github.com/JiNYiJY7/GPT_HEALTHMATE.git
   cd GPT_HEALTHMATE

2. **Install dependencies**
    ```bash
    pip install -r requirements.txt

3. **Run The Website**
    ```bash
    streamlit run app.py

4. **open in your browser**
    Local URL: http://localhost:8501
    Network URL: http://<your-ip>:8501
